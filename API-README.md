## API Reference

#### Get all items

```http
  GET /api/v1/lavalink
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `authorization` | `string` | **Required**. API Token |

#### Get item

```http
  GET /api/v1/version
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `Product`      | `string` | **Optional**. Project to fetch version from |
